\section{Summary}
\begin{itemize}
\item \colorbox{green!30}{Dark Green}: Very high speedup (≥100×)
\item \colorbox{green!15}{Light Green}: High speedup (50-100×)
\item \colorbox{yellow!20}{Yellow}: Moderate speedup (10-50×)
\item No color: Lower speedup (<10×)
\end{itemize}

\noindent \textbf{Key Observations:}
\begin{itemize}
\item CUDA provides substantial speedup over sequential CPU, especially for larger images
\item Tile size affects shared memory utilization and occupancy
\item Performance variation with tile size indicates memory access patterns and cache behavior
\item Larger images benefit more from GPU parallelization due to better amortization of overhead
\end{itemize}
